# Karim-Industries-MediTrack-Updated
Complete website with frontend already completed and going to start working on backend

## Project Overview
This project transforms a static React website into a full MERN stack application with advanced features including admin dashboards, real-time communication, and analytics.

## Features
- Complete MERN stack backend
- User and admin dashboards
- Real-time statistics and analytics
- Product CRUD operations with two-way communication
- Newsletter system
- Push notifications
- Advanced search and filtering
- Dynamic product management (migrated from hardcoded data)
- Webpage traffic analytics
- AI-powered chatbot
- Live contact messaging system

## Project Structure
- `website-project/` - React frontend application
- `skill.md` - Required technical skills and competencies
- `requirnment.md` - Detailed feature requirements and specifications
- `plan.md` - Comprehensive project plan and timeline

## Getting Started
1. Navigate to the website-project directory
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
